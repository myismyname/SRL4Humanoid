# SRL4Humanoid

The following code demonstrates how to train RL agents with SRL techniques on the open-source Unitree-G1-29dof-Velocity environment.

---

## 📁 Folder Structure

```
SRL4Humanoid/
├── srl4humanoid/              # Core SRL algorithm implementations
├── rsl_rl_lib/                # Core RL algorithm implementations
├── scripts/                   # Training and evaluation scripts
├── source/                    # Environment source code
├── robotic_assets/            # Robot URDF/USD files
```

---

## 🔧 Installation

- Users only need to install `IsaacLab==2.1.0` and `IsaacSim==4.5.0` to run the code.

---

See `run.sh` for all the training scripts.
